Major Recommendation System Installation Guide

This document provides a step-by-step guide on how to install Major Recommendation System.

Step 1: Download XAMPP

Download XAMPP from the following link. https://www.apachefriends.org/download.html.

Step 2: Install XAMPP

Locate the downloaded XAMPP installer file and run it. During the installation process, you will be asked to select the components you want to install and choose the installation directory (default recommended).

Step 3: Open the XAMPP Control Panel

You can run it from Start button or Task bar. Then start Apache and MySQL services.




Step 4: Download Python

Download Python from the following link: https://www.python.org/downloads/

Step 5: Install Python

Locate the downloaded Python installer file and run it. During the installation process, keep the default configuration parameters.

Step 6: install packages from the Python Package Index (PyPI) or other specified sources

Go to the command line CMD (run as administrator) and execute the following commands:

  • pip install requests
  • pip install beautifulsoup4
  • pip install PyQt5
  • pip install Flask
  • pip install flask-cors
  • pip install pandas
  • pip install openpyxl

Step 7: Place the application folder in the root of the web server

In the installation folder of XAMPP, you will find a sub-folder called htdocs . Copy the application folder of Major Recommendation System majors and paste it inside htdocs.

Step 8: Test the code

Run the command line CMD. Then change directory to majors that you have already placed inside htdocs. Run the following command:

  • python ex.py

Step 9: Launch the application

Open now your web browser, then write in the address bar the following expression localhost/majors

Step 10: Select/Input Data

In each screen (step), you are asked to select/input some data.ain the last phase, the system will suggest the suitable major(s) based on the inputs.
